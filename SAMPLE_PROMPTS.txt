CHROMEFORGE - SAMPLE TEST PROMPTS

Use these prompts to test chrome_forge.py
Simply run: python chrome_forge.py
Then enter any of these prompts when asked.

Optional: Gemini AI
-------------------
Gemini is used automatically for popup/content/style generation when `GEMINI_API_KEY`
is set. If you don't want Gemini, unset the environment variable or remove
`requirements['use_gemini']` in `chrome_forge.py`.

TEST CASE 1: Simple Popup Extension
================================================================================
PROMPT:
Create an extension that shows a popup with today's date.

EXPECTED OUTPUT:
- manifest.json
- popup.html
- popup.js
- styles.css

HOW TO TEST:
1. Run: python chrome_forge.py
2. Enter the prompt above
3. Open Chrome -> chrome://extensions/
4. Enable Developer mode
5. Click "Load unpacked"
6. Select "generated_extension" folder
7. Click the extension icon
8. Should see a styled popup showing today's date

QUICK REFERENCE - CHROME EXTENSION LOADING
================================================================================

1. Open Chrome
2. Navigate to: chrome://extensions/
3. Toggle "Developer mode" (top-right corner)
4. Click "Load unpacked"
5. Navigate to the "generated_extension" folder
6. Click "Select Folder"
7. Extension should appear in your extensions list
8. Click extension icon to test (if it has a popup)

================================================================================
TROUBLESHOOTING
================================================================================

Problem: Extension won't load
Solution: Check manifest.json is valid JSON, ensure all referenced files exist

Problem: Popup doesn't show
Solution: Verify manifest has "action" field with "default_popup"

Problem: Content script not working
Solution: Check permissions (activeTab, scripting) are in manifest

Problem: Background script errors
Solution: Check service_worker field in manifest, view errors in Extensions page

================================================================================
VERIFICATION CHECKLIST
================================================================================

After generating an extension:

[ ] generated_extension/ folder created
[ ] manifest.json exists and is valid JSON

[ ] Extension loads in Chrome without errors
[ ] Extension icon appears in toolbar (if popup)
[ ] Extension functionality works as described
